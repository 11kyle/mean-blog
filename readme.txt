Welcome to mean-blog, a blog built with node.js, express and mongoose.
Before the project can run, node.js and mongodb must be installed.

Install node.js

https://nodejs.org/en/download/

Install mongodb

https://docs.mongodb.com/manual/installation/

Follow the tutorial on mongodb's website to create the data directory
and set the permissions of the user to read and write the directory.

Run MongoDB

1. Open the terminal
2. Enter mongodb

Run the server.js

1. Open a new tab in the terminal
2. Go to the mean-blog directory by using the cd command
3. Type node server.js

Run the project
1. Open a browser
2. Go to localhost:3000

The project should now be running in the browser.
